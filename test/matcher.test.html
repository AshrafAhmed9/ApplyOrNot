ApplyOrNot — matcher.js / experience.js regression harness

Runs structural/gate logic against a fake bag-of-words embedder (no model download needed). This validates gating, weighting, and thresholds — it does not prove semantic paraphrase understanding (e.g. "Kubernetes" ~ "container orchestration"). Those "semantic wins" cases must be verified in the loaded extension against the real MiniLM embeddings (see comments below each fake-embedder-limited case).
